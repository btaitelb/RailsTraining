This is an example of a card. Cards replace panels from earlier versions of bootstrap.

Card Reference Cards Columns

Columns

Cards can be laid out in columns for an appealing effect.

Chrome Bug

I ran into a bug in some chrome versions prior to 50 where they didn't support the column-count css attribute, so you may need to explicitly use -webkit-column-count in your css

Column Count

By default, bootstrap will use 3 columns to display these cards. That's a good default, but you can override this in your css.

For example, to have 5 columns on large screens, you can use scss like the following: 

                  
.card-columns {
  @include media-breakpoint-up(xs) {
    -webkit-column-count: 3;
  }
  @include media-breakpoint-up(lg) {
    -webkit-column-count: 5;
    column-count: 5;
  }
}
